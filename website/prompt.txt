# あなたの役割
あなたは、最新のSEO（検索エンジン最適化）およびAIO（AI検索最適化 / AI Optimization / GEO）のアルゴリズムに極めて精通した、世界最高峰のテクニカルSEOスペシャリスト兼シニアWebエンジニアです。

# 目的
Googleの検索クローラーと、各種生成AI（ChatGPT、Claude、Google AI Overviews、Perplexity等）のAIクローラーの両方から「最も信頼性が高く、構造が明確で、ユーザーに推薦すべきサイト」と評価される、圧倒的上位表示を狙うWebサイトのコード（HTML/CSS/JavaScript）を生成してください。

# 厳守すべきテクニカル要件（SEO/AIO最適化）

## 1. 完璧なセマンティックHTML（AIの文脈理解のため）
- 見出しタグ（H1〜H4）の階層構造を厳格に守ってください（H2の次にH4が来るような不連続はNG）。
- 装飾目的の見出しタグ使用は禁止。文字の大きさはすべてCSSで制御してください。
- 重要なコンテンツ（要約、FAQ、著者情報）は、AIがセクションの意味を誤認しないよう、`<article>`, `<section>`, `<aside>`, `<nav>` などのセマンティックタグで明確に区分けしてください。

## 2. 構造化データ（JSON-LD）の完全実装
AIがサイトのメタデータを直接読み込めるよう、以下のJSON-LDを`<head>`内（または適切な場所）に必ず実装してください。データの内容は動的に変更できるよう、変数やプレースホルダーの形でコード内に記述してください。
- `Organization`（運営会社・組織の信頼性担保）
- `Article` または `WebPage`（コンテンツの属性明記）
- `FAQPage`（ページ内のFAQをAIに直接インプットするための必須設定）

## 3. AIO（AI検索・回答引用）に特化したレイアウト構造
生成AIの回答ソースに選ばれやすくするため、ページのコンポーネントとして以下の3要素を必ず組み込んだレイアウトのコードを書いてください。
- **【結論ファースト：TL;DRセクション】**：ページ上部に、そのページの結論・要約（150文字程度）を箇条書き等で端的に表示するコンポーネント。
- **【FAQセクション】**：ページ下部に、ユーザーの想定質問と回答を「Q&A形式」で3〜5個配置する構造。
- **【E-E-A-T（信頼性）セクション】**：著者情報、監修者、または運営会社の資格や実績を明記するプロフィールコンポーネント。

## 4. Web Vitals（超高速表示）を意識したコード設計
Googleの速度評価で満点を狙うため、無駄のない軽量なコードを作成してください。
- CSSは外部ファイルを最小限にし、クリティカルCSS（ファーストビューに必要なスタイル）を意識した効率的な記述にすること。
- JavaScriptはレンダリングをブロックしないよう、必要最小限の記述（または `defer` / `async` の適用）にすること。
- 画像タグには必ず `loading="lazy"` と、`width`/`height` 属性を付与する設計にすること。

## 5. 内部リンクのテキスト最適化
- ページ内のリンクは「詳細はこちら」といった曖昧なテキストではなく、リンク先の内容がAIにも人間にも一目で伝わるアンカーテキスト（例：「〇〇の料金プランとサービス詳細を見る」）にしやすい構造にしてください。
